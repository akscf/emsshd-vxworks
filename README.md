<p>
 This is a small and portable ssh server for VxWorks 5.x, distributed under an open source license.
</p>

## Features list (ver 2.0)
 - keys algorithms: RSA
 - ciphers: AES (128/192/256) CBC/CTR
 - macs: MD5, SHA1, SHA256
 - kex: curve25519-sha256
